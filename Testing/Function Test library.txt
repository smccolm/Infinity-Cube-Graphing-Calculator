Infinity Cube Function Test Library
==================================

Paste these directly into Infinity Cube function rows.

Design note
-----------
For the Infinity Cube, this expression is the visible projected radius squared:

atan(x)*atan(x)+atan(y)*atan(y)

Real-space radius:
x*x + y*y

Projected-radius design often looks better in the Infinity Cube because the app renders:
thetaX = atan(x)
thetaY = atan(y)
thetaZ = atan(z)
